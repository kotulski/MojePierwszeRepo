# MojePierwszeRepo

To jest moje pierwsze repozytorium na GitHubie.  
Zawiera przykładowy plik tekstowy `Witaj.txt`, który można otworzyć i przeczytać.

## 📄 Zawartość

- `Witaj.txt` – prosty plik tekstowy z przykładową zawartością.

## 🚀 Jak otworzyć plik

Aby otworzyć plik `Witaj.txt`, możesz:

### 🔹 Sposób 1: bezpośrednio na GitHubie
1. Kliknij nazwę pliku `Witaj.txt` w repozytorium.  
2. Zawartość pojawi się bezpośrednio w przeglądarce.

### 🔹 Sposób 2: lokalnie na komputerze
1. Pobierz repozytorium (kliknij **Code → Download ZIP** lub użyj `git clone`).  
2. Otwórz plik `Witaj.txt` w dowolnym edytorze tekstu, np. **Notepad**, **Notepad++**, **VS Code** lub **gedit**.

## 💡 Cel projektu

Repozytorium zostało utworzone w celu nauki podstaw korzystania z GitHuba — dodawania plików, commitów, oraz tworzenia pierwszego README.
